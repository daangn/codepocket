### Codepocket 구동하기

Codepocket은 docker compose파일을 제공하고 있음으로 쉽게 동작시켜 볼 수 있어요.

Docker Desktop을 설치해주고 아래 명령어를 실행시키면 쉽게 Codepocket을 동작시킬 수 있어요!

```
docker compose up -d
```
